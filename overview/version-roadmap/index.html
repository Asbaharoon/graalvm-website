Version Roadmap of Oracle GraalVM Enterprise Edition

Feature Release

Minor Feature releases of GraalVM Enterprise are produced every three months on a predictable schedule, always to the closest Tuesday to the 17th of the month of February, May, August and November. Once a new Feature release is made available, it will supersede any previous one.

Annual Release

There is an Annual release train for each calendar year numbered 19.x, 20.x, 21.x, etc.. Each Annual release train is comprised of a series of minor Feature releases. The final Feature release, e.g., 20.3, is supported for 18 months with bug and vulnerability fixes.

Critical Patch Update (CPU)

Critical Patch Updates (CPU) for GraalVM Enterprise follow the schedule for all CPU releases of Oracle as described here. The release happens quarterly always on the closest Tuesday to the 17th of the month of January, April, July, and October. All active releases receive patch updates.

Long-Term-Support (LTS) Release

A Long-Term-Support (LTS) release of GraalVM Enterprise will be designated every three years. Oracle GraalVM Enterprise Edition 20.3 will be the first LTS release. For Oracle GraalVM Enterprise Edition Support Polices, please see Oracle Lifetime Support Policy for Oracle Fusion Middleware Products.
