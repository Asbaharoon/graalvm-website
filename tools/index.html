GraalVM Enterprise Debugging and Monitoring Tools

GraalVM Enterprise provides a set of tools for developers to debug and monitor deployed applications or the GraalVM Enterprise platform itself. Traditional tools may not be ready to diagnose polyglot programs, while GraalVM Enterprise tools are implemented to inspect single or multi language applications by debugging via numerous clients, profiling, statement counting, dynamic analysis, and many more. These are:

  • VisualVM
  • Chrome Debugger
  • GraalVM Visual Studio Code Extensions
  • Language Server Protocol
  • GraalVM Insight
  • Code Coverage Command Line Tool
  • Profiling Command Line Tool
  • Ideal Graph Visualizer

GraalVM Enterprise also provides the Truffle Instrument API for creating language-agnostic tools like debuggers or profilers. To get started with a custom tool implementation on top of GraalVM Enterprise, proceed to the guide.
