GraalVM Enterprise Reference Manual

This reference manual provides a user of Oracle GraalVM Enterprise Edition with operational information, how-to instructions, available features and command line options in the release.

Disclaimer: Oracle GraalVM Enterprise Edition is developed on top of Oracle JDK, thus the documentation provided here contains only material that makes GraalVM Enterprise distinctive from the base.

Oracle GraalVM Enterprise Edition (GraalVM Enterprise) 20 is a high performance runtime for Java, JavaScript, LLVM-based languages such as C and C++, and other dynamic languages. The core GraalVM Enterprise distribution contains an optimizing compiler, JavaScript as a pre-installed extension language with Node.js runtime, and LLVM bitcode interpreter. This reference documentation instructs how to:

  • Run JVM-based Languages
  • Run JavaScript and Node.js
  • Run LLVM Languages
  • Run Python
  • Run R
  • Run Ruby
  • Run WebAssembly
