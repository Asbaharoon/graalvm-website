GraalVM Enterprise as a Platform

GraalVM Enterprise is an open ecosystem and allows to implement a custom language or tool on top of it with the Truffle Language Implementation framework.

Implement your own language

With the Truffle API you can implement a language on top of GraalVM Enterprise by automatically deriving high-performance code from interpreters.

Implement your own tool

With the Truffle Instrument API you can create language-agnostic tools like debuggers, profilers or other instruments on top of GraalVM Enterprise.
